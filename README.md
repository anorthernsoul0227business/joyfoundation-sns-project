# JoyFoundation SNS広報プロジェクト

サウンドヒーリング協会（NPO法人）と株式会社ジョイファンデーションのSNS広報・集客プロジェクト

## 概要

2つの関連団体の情報発信を統合的に管理し、Instagram、X、note、公式LINEなどを活用した集客を行う。

### 対象団体

| 団体名 | 種別 | 設立 | URL |
|--------|------|------|-----|
| サウンドヒーリング協会 | NPO法人 | 2001年 | https://www.sound-healing.jp/ |
| 株式会社ジョイファンデーション | 法人 | 1993年 | https://www.h-garden.com/ |

### 団体の関係性

- **サウンドヒーリング協会**: 理念・研究・教育を担当
- **ジョイファンデーション**: 製品化・商用サービスを担当

> 「ジョイファンデーション社はサウンドヒーリング協会が提唱する３つのメソッドを形にする企業」

## プロジェクト目標

1. SNSでの定期的な情報発信（Instagram、X、note）
2. 公式LINE構築と会員登録制の導入
3. 認知度向上と集客の実現
