# サウンドヒーリング協会・ジョイファンデーション 資料まとめ

## 基本情報

### 会社・団体の歴史
| 年 | 出来事 |
|---|---|
| 1986年 | 日野原重明先生が「日本バイオミュージック研究会」設立。喜田氏が当時勤務していた会社が関わる |
| 1993年 | HealingVibration.inc 設立（喜田圭一郎） |
| 1994年 | 事業スタート |
| 1999年 | 上野圭一氏の紹介でミッチェル・ゲイナー博士と出会う |
| 2001年 | サウンドヒーリング協会設立（ゲイナー博士の奨めで） |
| 2011年 | 社名を「ジョイファンデーション」に変更（喜びの田んぼ） |
| 2015年秋 | ミッチェル・ゲイナー博士 逝去 |
| 2017年7月 | 日野原重明先生 逝去（105歳） |
| 2021年 | サウンドヒーリング協会 設立20周年 |

## サウンドヒーリング協会の3つのメソッド

### 1. Harmonic Ambience（自然音）
- **目的**: 心と体に自然のリズムを取り戻す
- 故中田悟氏収録のマスターから編集
- 10,000Hz以上の高周波成分が特徴
- **1997年から累計20万人以上が活用**
- **収録地**: 屋久島、宮古島、石垣島、白神山地、富士山、霧島連山など
- 「音を見て、時を聴く」楽しみ方

### 2. Harmonic Massage（体感音響）
- **目的**: 体内環境を整える、細胞レベルのマッサージ
- **発案者**: 糸川英夫博士（日本のロケット開発の父）
- 150Hz以下の低音振動を骨伝導で伝える
- 宮沢賢治「セロ弾きのゴーシュ」の再現
- **研修は100回以上開催**
- **海外展開**: 米国、英国、カナダ、台湾、上海、タイ、韓国
- ロナルド・ピターズ博士（Mind Body Medicine Center）が医療に導入
- 「体の叡智（Wisdom of Body）」を働かせる

### 3. Divine Resonance（呼吸と声）
- **目的**: 高次元の響きに共鳴する
- ゲイナー博士の方法：「Infinite Harmony、私は健康」などを発声
- 「ことだま」の力を活用
- Voice Prayer（祈りの声）
