# ChatGPT用 資料整理プロンプト

## 使い方
1. ChatGPT-4oを開く
2. PDFファイルを10-20個アップロード
3. 下記プロンプトをコピペして送信
4. 出力を「資料まとめ.md」にコピペ

---

## プロンプト（コピペ用）

```
アップロードしたファイルを読み込み、以下の形式で構造化してください。

【出力形式】
各ファイルごとに以下のMarkdown形式で出力：

---

## [ファイル名]

**カテゴリ**: [学会発表/論文/体験談/解説/イベント報告/その他]
**年代**: [YYYY年]
**キーパーソン**: [登場する専門家・医師・研究者名]

### 概要
[3-5文で内容を要約]

### 主要データ・エビデンス
- [数値データや実験結果があれば箇条書き]
- [統計情報]
- [効果の具体例]

### キーフレーズ（SNS活用可能）
> 「[印象的な引用文1]」
> 「[印象的な引用文2]」

### SNS活用アイデア
- Instagram: [投稿アイデア]
- note: [記事アイデア]

---

【注意事項】
- 専門用語は一般向けに言い換えも併記
- 数値データは正確に抽出
- 人名・団体名は正確に記載
- 重複内容のファイルは「※[ファイル名]と同一内容」と記載
```

---

## カテゴリ別プロンプト（オプション）

### 学会発表・論文用
```
学会発表/論文PDFを読み込み、以下を抽出してください：

1. **発表情報**: 学会名、開催年、発表者
2. **研究目的**: 何を調べたか
3. **方法**: 被験者数、測定項目、使用機器
4. **結果**: 数値データ（p値、変化率など）
5. **結論**: 主張のポイント
6. **SNS向け要約**: 一般向けに100文字で

Markdown形式で出力してください。
```

### 体験談・感想用
```
体験談/感想ファイルを読み込み、以下を抽出：

1. **体験者属性**: 年代、性別、職業（わかれば）
2. **使用状況**: 何を、どのくらいの期間
3. **効果**: 具体的な変化
4. **印象的な言葉**: 引用可能なフレーズ
5. **信頼性**: 医師/専門家のコメントがあるか

Markdown形式で出力してください。
```

---

## 処理効率化のコツ

1. **同じカテゴリをまとめて処理**
   - 学会発表17本 → 2回で完了
   - 体験談20本 → 2回で完了

2. **重複チェック**
   - ファイル名が似ているものは同時にアップロード
   - ChatGPTが重複を検出してくれる

3. **出力の保存**
   - ChatGPTの出力をそのままコピー
   - 資料まとめ.mdの該当セクションにペースト
