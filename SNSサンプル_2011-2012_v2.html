note #1｜実践者インタビュー系
北海道から届いた「音の癒し」実践レポート
エステ×サウンドヒーリングで見えてきた「思考が現実を創る」という真実

「思考・感情が現実を創る」

この言葉を、あなたは信じますか？スピリチュアルな話に聞こえるかもしれません。でも、北海道でエステサロンを営む新谷香代さんは、この言葉を「身をもって体験した」と語ります。

日本最北のサウンドヒーリングサロン

新谷さんは、サウンドヒーリング協会のセラピスト資格を取得した10期生。北海道という、協会の拠点である東京から遠く離れた地で、エステの施術にサウンドヒーリングを取り入れる実践を始めました。

「最初は半信半疑でした」と新谷さんは振り返ります。「エステで体を整えることはできても、心までケアできるのか。音にそんな力があるのか、正直わかりませんでした」

お客様の変化、そして自分自身の変化

転機は、施術を重ねる中で訪れました。体感音響を取り入れたセッションを受けたお客様から、こんな声が届くようになったのです。

「体だけじゃなくて、心まで軽くなった気がする」
「なぜか涙が出てきた。でも、すっきりした」
「自分を責めていた気持ちが、少し和らいだ」

お客様の心身の変化を間近で見ているうちに、新谷さん自身にも変化が起きました。

新谷さんの気づき

「お客様の変化を見ていて、ふと気づいたんです。私自身も変わっていたことに。以前は『うまくいかないかも』と不安になることが多かったのに、いつの間にか『きっと大丈夫』と思えるようになっていた。『思考・感情が現実を創る』という言葉を、身をもって体験しました」

なぜ「音」が心に作用するのか

サウンドヒーリング協会の研究によれば、音が心身に作用するメカニズムには科学的な根拠があります。

人間の体は約70%が水で構成されています。音は振動であり、水は振動を伝えやすい媒体です。つまり、私たちの体は音の影響を非常に受けやすい構造になっているのです。

協会理事長の喜田圭一郎氏は、こう説明します。

「音の振動は縦の波で、水分子の溶解力を高めます。体に当てると、固執した思いを手放しやすくなる。古代より音の癒しは、宇宙に満ちる秩序と調和の力を地上とつなぐものとされてきました」

また、脳科学の観点からも音の効果は説明できます。2010年に開催された「第4回サウンドヒーリング研究会」では、脳科学者の有田秀穂氏が登壇し、セロトニンと音の関係について講演しました。

セロトニンは「幸せホルモン」とも呼ばれ、心の安定に深く関わる神経伝達物質。リズミカルな音や自然音は、このセロトニンの分泌を促進すると考えられています。

エステ×サウンドヒーリングという新しい形

新谷さんのサロン「サロン・ド・ボーテ・エクラ」では、エステの施術前後にサウンドヒーリングの要素を取り入れています。

  • 施術前：自然音を流しながら深い呼吸を促し、心身をリラックス状態へ
  • 施術中：体感音響を使い、体の深部まで振動を届ける
  • 施術後：静かな時間を設け、体の変化を感じてもらう

「エステで体を整え、サウンドヒーリングで心を整える。この組み合わせが、お客様の自己変容を促すことを、日々の施術の中で実感しています」と新谷さんは語ります。

「どこでも、誰でも」という希望

新谷さんの実践が示しているのは、サウンドヒーリングは特別な場所でなくても実践できるということです。

協会の拠点は東京ですが、北海道でも、富山でも、全国各地でセラピストたちが活動しています。同じく10期生の恒田由華さんは富山で「Total Salon VITA」を運営し、地域に根ざしたサウンドヒーリングの実践を続けています。

日本最北のサロンから届いたこのレポートは、「どこでも、誰でも、音の力を活用できる」という希望のメッセージでもあるのです。

  • 「思考が現実を創る」は実践者の体験で裏付けられている
  • 体の70%は水。音（振動）の影響を受けやすい構造
  • セロトニンと音の関係は脳科学でも研究されている
  • エステ×サウンドヒーリングという新しいアプローチ

note #2｜教育×ヒーリング
学習塾に「音の癒し」を導入してみた結果
子どもたちのストレスケアに、サウンドヒーリングができること

受験勉強、習い事、友人関係、SNS...

現代の子どもたちは、大人が想像する以上のストレスを抱えています。文部科学省の調査によれば、小中学生の不登校児童生徒数は年々増加し、2022年度には約29万人に達しました。

「成績を上げること」だけが教育の目的なのか。子どもたちの心のケアは、誰がするのか。学習塾「れもんの木」を運営する藤部久子さん（セラピスト11期生）は、この問いに向き合い続けてきました。

学習塾でサウンドヒーリング？

「最初は周りから不思議がられました」と藤部さんは笑います。「学習塾でヒーリング？何それ？って」

しかし藤部さんには、明確な問題意識がありました。

「塾に来る子どもたちの中に、明らかに疲れている子がいるんです。勉強以前に、心がいっぱいいっぱいになっている。その状態で詰め込んでも、効果は薄い。まず心を整えることが、学びの土台になるんじゃないかと思ったんです」
5分間の「音の時間」

藤部さんが導入したのは、授業前の5分間、自然音を聴く時間でした。

森の音、川のせせらぎ、鳥のさえずり。サウンドヒーリング協会が制作した自然音源を、教室に静かに流します。子どもたちには、目を閉じて、深い呼吸をするよう促します。

最初は落ち着かなかった子どもたちも、回を重ねるうちに変化が現れました。

子どもたちの変化
  • 「音の時間」の後、明らかに集中力が上がる
  • イライラしていた子が、穏やかな表情になる
  • 「今日も音聴ける？」と楽しみにする子が増えた
  • 保護者から「家でも落ち着いている」という報告
なぜ自然音が子どもの心を整えるのか

サウンドヒーリング協会の研究によれば、自然音には「概日リズム」が封じ込められています。

  • 朝の自然音: 体を活動モードにする
  • 昼の音: 心を穏やかにする
  • 夜の音: 体を休息モードにする

協会理事長の喜田圭一郎氏は、自然音の効果についてこう説明します。

「自然音は意識して聴かなくても、室内に流し、身体に浴びる感覚で聴くと良い。テレビや他の音楽を聞いていても、自然音はぶつかり合うことなく、室内が快適になります。目を閉じて視覚情報をストップし、自然音に意識を集中すると、五感の働きが高まり、直観力が磨かれます」

産業の発展とともに、私たちの周りには人工的なノイズが増えました。スマートフォンの通知音、車の騒音、エアコンの作動音...。子どもたちは、知らず知らずのうちにこうしたノイズにさらされ、無意識に体に力が入っています。

自然音は、そうした緊張を解きほぐし、本来の「安心する音」を思い出させてくれるのです。

「水と音で世界を平和に」

同じく協会で学んだ松尾千鶴さん（アイリ認定校セラピスト1期生）は、「水と音で世界を平和に」というテーマで活動しています。

水と音と振動による感情解放のワーク。一見、教育とは関係なさそうですが、松尾さんはこう語ります。

「子どもたちの問題行動の多くは、感情の抑圧から来ています。怒り、悲しみ、不安...。それらを安全に解放する場があれば、子どもたちはもっと楽になれる。音は、言葉にできない感情を解放する助けになるんです」
教育の新しい形

学校でも塾でも、「成績」という結果が求められる場所。でも本当に大切なのは、子どもたちが安心して学べる環境ではないでしょうか。

サウンドヒーリング協会は2001年の設立以来、「音を変えれば人が変わる、人が変われば社会が変わる」という理念を掲げてきました。

藤部さんの実践は、その理念を教育現場で具現化したものと言えます。「学ぶ」と「癒される」が共存する空間づくり。それは、これからの教育に必要な視点かもしれません。

  • 学習の前に「心を整える」ことが効果的

note #3｜科学的エビデンス系
「体感音響」の効果を学会で発表しました
第76回日本温泉気候物理医学会での研究報告

「サウンドヒーリングって、本当に効果があるの？」

この疑問は、私たちが最もよく受ける質問のひとつです。「なんとなく気持ちいい」「リラックスする気がする」...でも、それって気のせいじゃないの？プラセボ効果じゃないの？

この疑問に科学的に答えるため、サウンドヒーリング協会では20年以上にわたり、継続的な効果測定と学会発表を行っています。

第76回日本温泉気候物理医学会での発表

2012年、協会理事長の喜田圭一郎は、日本温泉気候物理医学会で「体感音響が与える体と心への影響」と題した研究発表を行いました。

体感音響とは、音を「聴く」だけでなく、振動として「体で感じる」システムです。専用の機器を通じて、音楽や自然音の振動を体の深部まで届けます。

この研究では、体感音響セッションの前後で、複数の生理指標を測定しました。

体感音響セッション前後の変化（測定データ）
顔面水分量 32.7% → 40.4%（+23.5%）
手掌温度 29.67℃ → 31.42℃（+1.75℃）
足部温度 21.8℃ → 24.3℃（+2.5℃）
状態不安指数（STAI） 39.52 → 28.28（-28.4%）
このデータが意味すること

これらの数値は、何を意味しているのでしょうか。

1. 肌の水分量が上がった（32.7%→40.4%）

肌の水分量は、自律神経の状態と密接に関係しています。緊張状態では血管が収縮し、肌への血流が減少して水分量が低下します。逆に、リラックス状態では血管が拡張し、肌への血流が増加して水分量が上がります。

2. 末端の温度が上がった（手+1.75℃、足+2.5℃）

「冷え性」に悩む方は多いですが、末端の冷えは自律神経の乱れのサインとも言われます。体感音響後に手足の温度が上昇したことは、副交感神経が優位になり、末端まで血流が行き届くようになったことを示唆しています。

3. 状態不安指数が下がった（39.52→28.28）

STAI（State-Trait Anxiety Inventory）は、心理学で広く使われる不安測定尺度です。「今、この瞬間」の不安レベルを数値化します。28%以上の低下は、統計的にも有意な変化と言えます。

なぜ「体で聴く」ことが重要なのか

一般的な音楽療法と体感音響の違いは、「耳で聴く」だけでなく「体で聴く」点にあります。

人間の体は約70%が水で構成されています。音は振動であり、水は振動を伝えやすい媒体です。体感音響では、この原理を利用して、音の振動を体の深部まで届けます。

「音の振動は縦の波で振動し、水分子の溶解力を高める。体に当てると固執した思いを手放しやすくなる」
──喜田圭一郎（サウンドヒーリング協会理事長）

協会のアドバイザーであるロナルド・ピーターズ博士（米国）は、現代の多くの病気は「生活スタイルからくる文明病（慢性疾患）」であり、薬で症状を抑えるだけでは根本的に治らないと指摘します。

博士が挙げる「病気を予防し根本から治す4つのポイント」は：

  • 毎日の食事
  • 体を動かす
  • デトックス
  • ストレスケア

喜田理事長はこれに加えて、「呼吸法」「肯定的な言葉」「音（自然音、体感音響）」を実践し、「30年以上、病院のお世話にならずにいます」と語ります。

「感覚」と「科学」の両立

サウンドヒーリングの世界では、「感覚」を大切にします。数値では測れない、言葉では表現しにくい「心地よさ」「安心感」「解放感」。それらは確かに存在し、体験した人には伝わります。

一方で、「それって本当なの？」という疑問に答えるためには、科学的なエビデンスも必要です。

私たちは、「感覚」と「科学」の両方を大切にしながら、音の可能性を探求し続けます。
