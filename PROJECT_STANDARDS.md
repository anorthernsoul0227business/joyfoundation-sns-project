| **タスクシート_YouTube.md** | YouTube専用タスク | 動画制作時 |

## タスク管理ルール

### Phase管理

各タスクシートはPhase単位で管理:

```markdown
## Phase X: タスク名（Week X-X）

| タスク | 完了条件 | チェック |
|--------|----------|----------|
| タスク1 | 条件1 | [ ] |
| タスク2 | 条件2 | [ ] |
```

### ステータス表記

| 記号 | 意味 |
|------|------|
| [ ] | 未着手 |
| [x] | 完了 |
| [!] | 進行中 |
| [-] | 保留/中止 |

### KPI管理

目標値は月次で設定し、達成状況を記録:

```markdown
## KPI目標（月次）

| 指標 | 目標 | 実績 | 達成率 |
|------|------|------|--------|
| Xフォロワー | 500 | - | - |
```

## コンテンツ制作ルール

### 命名規則

| 種類 | 形式 | 例 |
|------|------|-----|
| Instagram投稿 | IG_YYYYMMDD_カテゴリ_番号 | IG_20250301_科学_01 |
| X投稿 | X_YYYYMMDD_番号 | X_20250301_01 |
| YouTube動画 | YT_カテゴリ_番号 | YT_BGM_01 |
| note記事 | NOTE_YYYYMMDD_タイトル | NOTE_20250301_自律神経 |

### カテゴリ分類

| カテゴリ | 内容 | 活用プラットフォーム |
|----------|------|----------------------|
| 科学・エビデンス | 研究データ、実験結果 | 全SNS |
| 哲学・理念 | 理事長メッセージ、名言 | Instagram、X、note |
| 実践・Tips | 聴き方、呼吸法 | Instagram、YouTube |
| イベント | 告知、レポート | 全SNS |
| ストーリー | 歴史、人物 | note、YouTube |

## Gitコミット規約

### コミットメッセージ形式

```

Co-Authored-By: Claude Opus 4.5 <noreply@anthropic.com>
```

### タイプ

| タイプ | 用途 |
|--------|------|
| feat | 新規コンテンツ追加 |
| fix | 修正 |
| docs | ドキュメント更新 |
| content | SNS投稿作成 |
