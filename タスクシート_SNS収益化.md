# タスクシート：SNS収益化プロジェクト

## プロジェクト概要
- **目標**: SNSを活用した認知拡大・収益化の実現
- **対象**: サウンドヒーリング協会 / ジョイファンデーション
- **期間**: 3ヶ月（Phase 1-5）
- **メインターゲット**: 40代〜60代女性（子育てを終えた層）
- **核心メッセージ**: 病院に頼らず「自己回復力」を高める知恵を届ける

## 担当プラットフォーム
1. X（Twitter）- 毎日投稿
2. Instagram - 週4回
3. YouTube - 週2回
4. note - 週1回
5. 公式LINE - 週1回配信

---

## 最終ゴール

SNSからの流入で月間収益15万円以上を達成し、正会員・イベント参加者を継続的に獲得できる状態

---

## 長期目標（代表・圭一郎さんとの会話より）

### ビジョン
**資格を取得する実践者（セラピスト）を継続的に育成し、5年〜10年の仲間として活動できる人材を増やす**

### 段階的ゴール
```
【Phase 1-5】認知獲得・収益化（現在のタスク）
Instagram/SNS → 初めての人に協会を知らせる
     ↓
【Phase 6-7】体験誘導（今後の目標）
オンラインワークショップ / Harmonic Day / リアル体験会
     ↓
【Phase 8-9】継続学習コミュニティ
定期的なスキルアップ勉強会 → 核となる実践者を育成
     ↓
【Phase 10】資格取得者育成
認定セラピスト制度 → 家族や地域を支える人材輩出
```

### ブランディング方針
| 課題 | 対策 |
|------|------|
| 「サウンドヒーリング」が怪しいと思われがち | 科学的・歴史的根拠を前面に出す |
| 信頼性の担保 | 実際の体験談を許可を取って紹介 |
| 医療表現への注意 | 「治療」ではなく「回復力を高める」表現 |
| 既存コンテンツ活用 | 過去の出版物をnote等でデジタル化 |
